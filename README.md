# Meu Blog - Página de Criação de Posts

## Descrição
Este é um projeto simples de uma página web para criação e visualização de posts em um blog. Permite aos usuários criar novos posts através de um formulário, que são enviados para uma API simulada (JSONPlaceholder) e exibidos na página com feedback visual.

## Funcionalidades
- Formulário para criação de posts com título e conteúdo
- Envio assíncrono para API simulada
- Exibição do post criado com metadados (data, autor)
- Feedback visual: loading, sucesso e erro
- Interface responsiva e moderna
- Botões de ação simulados (curtir, comentar, compartilhar)

## Como Baixar e Instalar
1. **Clone ou baixe o repositório:**
   - Faça o download dos arquivos `index.html`, `style.css` e `script.js` para uma pasta no seu computador.

2. **Abra no navegador:**
   - Abra o arquivo `index.html` diretamente no seu navegador web (Chrome, Firefox, etc.).
   - Não há necessidade de servidor local, pois é uma aplicação estática.

## Uso
- Preencha o título e o conteúdo do post no formulário.
- Clique em "Publicar Post".
- Aguarde o feedback de loading e veja o post aparecer na página.
- Os posts são simulados e não persistem após recarregar a página.

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)

## API Utilizada
- [JSONPlaceholder](https://jsonplaceholder.typicode.com/) para simulação de posts.
